# AI Application Generator

This demo project generates a tailored motivation letter using:

- Python
- OpenAI API
- Streamlit UI
- Structured JSON profile data

## How it works

User enters a job title and job description.

The system combines:
- job description
- profile.json data
- prompt template

Then the AI generates a tailored motivation letter.

## Run locally

Install dependencies

pip install -r requirements.txt

Run the app

streamlit run app.py
